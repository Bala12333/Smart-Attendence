## UI Style Guide — Smart Attendance System

### Palette
- Indigo: `#4F46E5` (primary), `#4338CA` (hover), `#EEF2FF` (tint)
- Cool Gray: `#111827` (text), `#6B7280` (muted), `#E5E7EB` (borders), `#F3F4F6` (surfaces)
- White: `#FFFFFF`
- Semantic: Success `#16A34A`, Error `#DC2626`, Warning `#F59E0B`

### Typography
- Headings: Inter, weight 600–700
- Body: Inter, weight 400–500
- Scale: 12, 14, 16, 18, 20, 24, 30, 36

### Components
- App Shell: Top bar with app title; responsive sidebar for role navigation.
- Cards: White surface, rounded-lg, shadow-sm, border `#E5E7EB`, padding 16–24px.
- Buttons: Primary indigo solid; secondary ghost with indigo text; disabled at 60% opacity.
- Status Chips: Rounded-full; green for Present, red for Absent.
- Inputs: Rounded-md, border `#E5E7EB`, focus ring indigo.

### Layout — Lecturer Dashboard
- Left: Course selector dropdown (sticky).
- Center: QR Code generator/display card (prominent).
- Right: Pie chart card with overall attendance %.
- Bottom/Center: Live student list with status.

### Iconography
- Heroicons outline recommended.

### Spacing
- 8px base grid. Use 16/24 for block spacing.
